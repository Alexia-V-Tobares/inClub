# 🎟️ inClub: Gestión de Accesos Inteligente

![Python](https://img.shields.io/badge/Python-3.10+-3776AB?style=for-the-badge&logo=python&logoColor=white)
![AI Assisted](https://img.shields.io/badge/Developed%20with-Trae%20AI-6366f1?style=for-the-badge)

**inClub** es una solución integral para el registro de asistentes en establecimientos. El sistema combina una lógica robusta en el backend con una interfaz de usuario moderna desarrollada con asistencia de IA.

## 🚀 Innovación con IA (Trae)
Este proyecto fue desarrollado utilizando el IDE **Trae**, aprovechando sus capacidades de autocompletado inteligente y generación de código para:
- Diseñar una interfaz de usuario (Frontend) intuitiva y responsiva.
- Optimizar la conexión entre la lógica de Python y los elementos visuales.
- Refactorizar funciones para mejorar la eficiencia del código.



## ✨ Características Principales
- **Validación Automática:** Filtro de mayoría de edad y verificación de tickets.
- **Persistencia de Datos:** Carga y guardado automático en archivos de texto para no perder registros.
- **Registro Temporal:** Cada entrada incluye automáticamente fecha y hora exacta mediante el módulo `datetime`.

## 🛠️ Tecnologías Utilizadas
- **Lenguaje:** Python 3.x(Arquitectura modular con `funciones.py` y `main.py`).
- **Frontend:** Diseñado y optimizado utilizando la IA **Trae**, logrando una interfaz intuitiva.
- **Persistencia:** Manejo de archivos planos para el histórico de asistentes.
